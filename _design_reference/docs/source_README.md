# SuitUP - Sartorial Excellence

Uma aplicação Android de luxo desenvolvida em Kotlin com Jetpack Compose, baseada no design system "Sartorial Excellence".

## 🚀 Funcionalidades

- **Design de Precisão**: Interface focada em luxo, utilizando as fontes Playfair Display e Hanken Grotesk.
- **Editor 3D (Simulado)**: Fluxo de personalização de ternos com seleção de tecidos e visualização progressiva.
- **Catálogo Exclusivo**: Navegação por coleções de ternos premium.
- **Acompanhamento de Pedidos**: Gestão de status de produção e entrega.
- **Navegação Fluida**: Fluxo completo desde o login até a finalização do design.

## 🛠️ Tecnologias Utilizadas

- **Kotlin**: Linguagem principal.
- **Jetpack Compose**: UI declarativa moderna.
- **Material Design 3**: Base do sistema de componentes, customizada para o estilo "Quiet Luxury".
- **Compose Navigation**: Gestão de rotas e fluxos da aplicação.
- **Gradle (Kotlin DSL)**: Automação de build.

## 🎨 Design System

O projeto segue rigorosamente o arquivo `DESIGN.md`:
- **Cores**: Paleta Dark Mode com acentos em Ouro (#F2CA50) e Tabaco (#EEBD8E).
- **Tipografia**: Contraste entre Serif (Heritage) e Sans-Serif (Utility).
- **Componentes**: Botões estruturados, inputs minimalistas e o exclusivo "Progress Rail".

## 📂 Estrutura do Projeto

- `ui/theme`: Definições de cores, tipos, formas e espaçamento.
- `ui/components`: Componentes atômicos reutilizáveis.
- `ui/screens`: Telas completas da aplicação.
- `data/model`: Camada de dados e modelos simulados.
